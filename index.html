Research

Undergraduate Thesis

Title: Development of Bamboo–Jute Fiber Reinforced Epoxy Hybrid Composites Modified with Nanoclay Nanoparticles for Enhanced Mechanical, Thermal, and Moisture Resistance Properties.

Research Objectives

  • To improve the mechanical properties of natural fiber-reinforced epoxy composites.
  • To enhance thermal stability using nanoclay modification.
  • To reduce moisture absorption behavior of bamboo–jute hybrid composites.
  • To explore sustainable and biodegradable composite material applications.

Engineering Projects

Smart Watering System

Designed and developed an IoT-based Smart Watering System using the Seeed Studio XIAO ESP32-C6 microcontroller, soil moisture sensing, rain detection, OLED monitoring, relay-based pump control, and Blynk IoT cloud integration. The system automatically irrigates plants based on soil moisture conditions while allowing real-time remote monitoring and control.

  • Automatic irrigation control
  • Real-time soil moisture monitoring
  • Rain detection integration
  • Remote monitoring through Blynk IoT
  • OLED-based local display interface
  • Manual and automatic operating modes

RFID-Based Door Lock System

Designed a security access system using Arduino, RFID module, keypad, relay, LCD display, and solenoid lock.

  • RFID-based authentication
  • Password-based access control
  • LCD user interface
  • Relay-controlled locking mechanism

Gas Leakage Detection System

Developed a gas leakage detection system using MQ-2 sensor, Arduino, GSM module, buzzer, LEDs, and LCD display.

  • Gas concentration detection
  • GSM-based alert system
  • Visual and audio warning system
  • Emergency safety monitoring
